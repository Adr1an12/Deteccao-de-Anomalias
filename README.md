# Detecção de Anomalias em Logs de Acesso com Isolation Forest

Este projeto tem como objetivo identificar acessos potencialmente anômalos em logs de sistemas utilizando técnicas de Inteligência Artificial e Aprendizado de Máquina. Foi aplicado o algoritmo Isolation Forest, um método de aprendizado não supervisionado amplamente utilizado para detecção de anomalias.

O conjunto de dados contém informações sobre sessões de acesso ao sistema, incluindo horário de acesso, duração da sessão, número de tentativas de login, utilização de IP diferente do habitual e quantidade de páginas acessadas.

O projeto contempla as seguintes etapas:

* Análise exploratória dos dados;
* Pré-processamento e normalização das variáveis;
* Treinamento do modelo Isolation Forest;
* Identificação de acessos anômalos;
* Visualização dos resultados por meio de gráficos;
* Interpretação dos padrões encontrados;
* Discussão sobre possíveis riscos de segurança e limitações do modelo.

Os resultados demonstram como técnicas de detecção de anomalias podem auxiliar sistemas de monitoramento na identificação de comportamentos fora do padrão, contribuindo para a detecção precoce de atividades suspeitas e apoio à tomada de decisão em ambientes corporativos.
